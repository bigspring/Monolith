Monolith

a lightweight WordPress theme powered by Twitter Bootstrap, built for developers.

Download this project as a .zip file Download this project as a tar.gz file

Monolith is a starter WordPress theme built with Twitter Bootstrap. We started this project as our own internal starter theme, because most of the other theme frameworks out there either did too much or not enough for our needs. So we built our own.

Alongside the Bootstrap grid magic, monolith comes bundled with a few useful functions and shortcodes that we've found useful enough to use on most projects. Anything that is too project-specific or heavy has been stripped out, and can be easily implemented on a case by case basis.
